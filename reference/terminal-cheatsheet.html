Terminal Cheat Sheet

The 20 commands that cover 95% of daily work.

Orientation

CommandMnemonicWhat it does
pwdprint working directorywhere am I?
lslistwhat's here? (-l details, -a hidden files)
cd pathchange directorygo there (cd alone = home, cd .. = up, cd - = back)

Paths

SymbolMeans
/the root — top of the whole filesystem
~your home: /home/amjad
.the current directory
..the parent directory
starts with /absolute path (works from anywhere)
anything elserelative path (from where you are now)

Files & folders

CommandWhat it does
mkdir namemake directory (-p creates nested paths)
touch filecreate an empty file
cp src destcopy (-r for folders)
mv src destmove — also how you rename
rm filedelete — no trash, no undo (-r for folders)
cat fileprint file contents
less filepage through a file (q to quit)
head / tailfirst / last 10 lines (-n 25 for 25)

Finding things

CommandWhat it does
grep -r "text" .search file contents recursively
find . -name "*.html"search file names
man cmd / cmd --helpthe manual — always available offline

Superpowers

TrickWhat it does
Tabautocomplete paths/commands — use constantly, it prevents typos
↑ / Ctrl-Rprevious commands / search command history
Ctrl-Ckill the running command
a | bpipe: feed output of a into b — ls | wc -l counts files
a > fileredirect output into a file (>> appends)
*wildcard: rm *.tmp, ls week-*

Glossary

TermMeaning
terminalthe window program that displays text I/O
shellthe program inside it that interprets your commands (yours: bash)
promptthe text before your cursor, waiting for input
argument / flagextra words after a command; flags start with - and modify behavior
directorya folder — same thing, terminal dialect
