JavaScript Cheat Sheet

Week 1 scope: values, variables, operators, arrays, objects. Look up, don't memorize.

Declarations

KeywordUse
constdefault — no reassignment (mutating the object it points to is still allowed)
letonly when reassignment is genuinely needed
varlegacy — you'll read it in older code, don't write it

Types & comparisons

ExpressionResult / rule
typeof x"string" | "number" | "boolean" | "undefined" | "object" | "function"
typeof null"object" — historical bug, kept forever
=== / !==strict: same type AND value. Always use these
== / !=loose: converts types first ("5" == 5 → true). Avoid
[1] === [1]false — objects/arrays compare by reference, not content
falsy valuesfalse 0 "" null undefined NaN — everything else is truthy

Logical operators (they return an operand)

ExpressionReturns
a && ba if falsy, else b
a || ba if truthy, else b — classic fallback: name || "anonymous"
a ?? bb only when a is null/undefined (safer fallback: keeps 0 and "")
!aboolean negation
