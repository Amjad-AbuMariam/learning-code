# 0001 — Workspace initialized; starting point assessed

Date: 2026-07-13
Status: active

## Context

Amjad begins the GSG curriculum with prerequisites already done: freeCodeCamp
basics (including DOM manipulation) and Codewars 5kyu. He has a rusty CS degree,
so theory (data structures, complexity) needs reviving only on demand, not
teaching from scratch. Time budget 20+ h/week (initially reported as ~5,
corrected same day) — full bootcamp pace is feasible.

## Decision

Skip re-teaching JS/DOM basics. Enter at curriculum Week 1, but focus on the
gap freeCodeCamp leaves: professional workflow (git branching, PRs, code
review) and building real projects. Lesson 0001 covers the branch-and-PR loop.

## Consequences

- Pace: target ~1 curriculum week per calendar week; lessons stay short but
  each assigns real project work (that's where the 20 h go).
- Zone of proximal development: finish Week 1 (semantic HTML/flexbox, code
  review, array methods) this week, including the Week 1 project.
- If lessons feel too easy, jump faster toward Week 2–3; his floor is higher
  than a typical bootcamp entrant.
- Verify retention of git workflow with retrieval practice in a later lesson
  (spacing), before Week 2.
